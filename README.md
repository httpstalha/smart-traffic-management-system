
# Smart Traffic Management System 🚦

AI-powered Computer Vision project for smart traffic monitoring and analysis.

## Features
- Vehicle Detection
- Vehicle Counting
- Traffic Density Monitoring
- Lane Monitoring
- Real-time Webcam/CCTV Support
- Smart Traffic Signal Logic
- Traffic Analytics

## Technologies
- Python
- OpenCV
- YOLOv8
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python main.py
```
